    Operations

    There are various editor functions and settings to be aware of. To do this we will go through the top editor menu from left to right.

    First we see the 'Undo' and 'Redo' buttons, which work for all non-automated edits that you can make in the label editor.

    Next we have the bulk operations button. Here we can run deletion over multiple objects and frames across the video, as well as filter objects generated from automated labelling by confidence score.

    After this we have the label status button. Once you want to stop annotating a video, you can set the status of the video with this button. Options include 'Not labelled', 'In progress', 'Labelled','Review required', and 'Reviewed'.

    The next buttons are for 'Save', 'Zoom', and 'Brightness and contrast'. The 'Save' button allows you to save your labelling progress at any time. Labels will also auto-save at on a periodic time interval which you can set in the editor settings. Adjusting the 'Zoom', 'Brightness', and 'Contrast' allow you to change the view of the video.

    One of the most useful tools for efficient annotation is the effective use of hotkeys. All available hotkeys can be viewed by clicking on the next keyboard icon in the 'Hotkey shortcut' button.

    This menu will update as you add new labels in your video and will be a useful guide to get familiar with.

    Finally we have the editor settings button. These settings allow you to change the default tracking parameters and the auto-save time interval.
