    We can manage our data through the 'Datasets' tab. Here we see an overview of the datasets we have currently available.

    A dataset is collection of videos arranged together. If we enter one of the datasets we see that we can edit the dataset by removing videos or by uploading new videos to include within the dataset. You can also download and view individual videos by clicking the blue download icon at the right of each video.

    We can delete an entire dataset under the 'Settings' tab.
